### Task 2: Solar Production Time-Series Panel (PROD-01 + PROD-04)
- **Panel type:** `timeseries`
- **ID:** 12
- **Title:** "Solar Production"
- **Description:** "Power output over time — East (orange) vs West (blue) inverters"
- **gridPos:** `{h: 10, w: 16, x: 0, y: 10}`
- **Queries:**
  - Query A (refId: "East"): 
    ```sql
    SELECT DATE_BIN(INTERVAL '5 minutes', time) AS time, AVG(power_sensor) AS "East" 
    FROM "East Microinverter" 
    WHERE time >= $__timeFrom AND time <= $__timeTo 
    GROUP BY 1 ORDER BY 1
    ```
  - Query B (refId: "West"):
    ```sql
    SELECT DATE_BIN(INTERVAL '5 minutes', time) AS time, AVG(power_sensor) AS "West" 
    FROM "West Microinverter" 
    WHERE time >= $__timeFrom AND time <= $__timeTo 
    GROUP BY 1 ORDER BY 1
    ```
- **Format:** `time_series`
- **Field config:**
  - defaults: unit `"watt"`, min 0, custom fillOpacity 20, gradientMode "scheme"
  - overrides: East → color `#FF9830`, West → color `#5794F2`
- **Options:** tooltip mode "multi", legend displayMode "table", legend calcs ["mean", "max", "lastNotNull"]

### Task 3: Per-Inverter Current Power Bar Gauge (PROD-02)
- **Panel type:** `bargauge`
- **ID:** 13
- **Title:** "Inverter Power"
- **Description:** "Current power output per inverter"
- **gridPos:** `{h: 5, w: 8, x: 16, y: 10}`
- **Queries:**
  - Query A (refId: "East"):
    ```sql
    SELECT power_sensor AS "East" FROM "East Microinverter" ORDER BY time DESC LIMIT 1
    ```
  - Query B (refId: "West"):
    ```sql
    SELECT power_sensor AS "West" FROM "West Microinverter" ORDER BY time DESC LIMIT 1
    ```
- **Format:** `table`
- **Transformations:** merge
- **Field config:**
  - defaults: unit `"watt"`, min 0, max 2620, thresholds [0: dark-gray, 1: yellow, 500: light-green, 1500: green, 2200: dark-green]
  - overrides: East → color `#FF9830`, West → color `#5794F2`
- **Options:** orientation "horizontal", displayMode "gradient"

### Task 4: East vs West Production Split Pie Chart (PROD-02 enhancement)
- **Panel type:** `piechart`
- **ID:** 14
- **Title:** "East / West Split"
- **Description:** "Today's production split between East and West inverters"
- **gridPos:** `{h: 5, w: 8, x: 16, y: 15}`
- **Queries:**
  - Query A (refId: "East"):
    ```sql
    SELECT MAX(today_production_1_sensor) + MAX(today_production_2_sensor) + MAX(today_production_3_sensor) + MAX(today_production_4_sensor) AS "East" 
    FROM "East Microinverter" 
    WHERE time >= now() - INTERVAL '1 day'
    ```
  - Query B (refId: "West"):
    ```sql
    SELECT MAX(today_production_1_sensor) + MAX(today_production_2_sensor) + MAX(today_production_3_sensor) + MAX(today_production_4_sensor) AS "West" 
    FROM "West Microinverter" 
    WHERE time >= now() - INTERVAL '1 day'
    ```
- **Format:** `table`
- **Transformations:** merge, reduce (sum)
- **Field config:**
  - defaults: unit `"kwatth"`
  - overrides: East → color `#FF9830`, West → color `#5794F2`
- **Options:** pieType "donut", legend placement "right"
