## Tasks

### Task 1: Daily Energy Production Bar Chart (PROD-03)
- **Panel type:** `barchart`
- **ID:** 15
- **Title:** "Daily Energy"
- **Description:** "Daily energy production — East (orange) vs West (blue)"
- **gridPos:** `{h: 8, w: 24, x: 0, y: 20}`
- **Queries:**
  - Query A (refId: "East"):
    ```sql
    SELECT date_trunc('day', time) AS time,
           MAX(today_production_1_sensor) + MAX(today_production_2_sensor) + MAX(today_production_3_sensor) + MAX(today_production_4_sensor) AS "East"
    FROM "East Microinverter"
    WHERE time >= $__timeFrom AND time <= $__timeTo
    GROUP BY 1 ORDER BY 1
    ```
  - Query B (refId: "West"):
    ```sql
    SELECT date_trunc('day', time) AS time,
           MAX(today_production_1_sensor) + MAX(today_production_2_sensor) + MAX(today_production_3_sensor) + MAX(today_production_4_sensor) AS "West"
    FROM "West Microinverter"
    WHERE time >= $__timeFrom AND time <= $__timeTo
    GROUP BY 1 ORDER BY 1
    ```
- **Format:** `table` (barchart needs table format)
- **Transformations:** merge (combines East and West into one frame by time)
- **Field config:**
  - defaults: unit `"kwatth"`, min 0
  - overrides: East → color `#FF9830`, West → color `#5794F2`
- **Options:** stacking "normal", barWidth 0.7, xField "time", orientation "auto"

### Task 4: Grid Parameters Time-Series (GRID-02) — 2 panels
- **Panel 20: Voltage & Frequency Trends**
  - Type: `timeseries`, gridPos: `{h: 8, w: 12, x: 0, y: 33}`
  - Queries:
    - A: `SELECT DATE_BIN(INTERVAL '5 minutes', time) AS time, AVG(voltage_sensor) AS "Voltage (V)" FROM "Smart Meter" WHERE time >= $__timeFrom AND time <= $__timeTo GROUP BY 1 ORDER BY 1`
    - B: `SELECT DATE_BIN(INTERVAL '5 minutes', time) AS time, AVG(frequency_sensor) AS "Frequency (Hz)" FROM "Smart Meter" WHERE time >= $__timeFrom AND time <= $__timeTo GROUP BY 1 ORDER BY 1`
  - Format: `time_series`
  - Overrides: Voltage on left Y-axis (unit: volt), Frequency on right Y-axis (unit: hertz)

- **Panel 21: Power Factor & Current Trends**
  - Type: `timeseries`, gridPos: `{h: 8, w: 12, x: 12, y: 33}`
  - Queries:
    - A: `SELECT DATE_BIN(INTERVAL '5 minutes', time) AS time, AVG(power_factor_sensor) AS "Power Factor" FROM "Smart Meter" WHERE time >= $__timeFrom AND time <= $__timeTo GROUP BY 1 ORDER BY 1`
    - B: `SELECT DATE_BIN(INTERVAL '5 minutes', time) AS time, AVG(current_sensor) AS "Current (A)" FROM "Smart Meter" WHERE time >= $__timeFrom AND time <= $__timeTo GROUP BY 1 ORDER BY 1`
  - Format: `time_series`
  - Overrides: Power Factor on left Y-axis (unit: none, decimals 3), Current on right Y-axis (unit: amp)
