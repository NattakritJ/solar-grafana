<objective>
Replace all point-in-time latest-value queries (selector_last + ORDER BY DESC LIMIT 1) with windowed aggregates (AVG/MAX) to fix cross-device temporal misalignment.

Purpose: Each device logs at a different time and interval (inverters ~10s, CT meters ~2s). Using `selector_last` or `LIMIT 1` returns values from different instants, causing cross-device expressions (Self-Consumption, House Load) to combine non-contemporaneous readings. `AVG` over the existing 2-minute window aligns all devices to the same temporal window.

Output: Updated `solar-pv-monitor.json` with ~84 SQL query changes — zero panel layout/config changes.
</objective>
