**Panel 1: Total Solar Power (OVER-01)** — id: 1
- `type`: "stat"
- `title`: "Solar Power"
- `description`: "Current total solar production from both inverters"
- `gridPos`: { h: 4, w: 3, x: 0, y: 1 }
- `transparent`: true
- `datasource`: { "type": "influxdb", "uid": "${DS_INFLUXDB_SOLAR}" }
- `targets`: Two queries that sum both inverters (per D-11 — use power_sensor, NOT pv channel sums):
  - Query A (refId: "A"):
    ```sql
    SELECT COALESCE(power_sensor, 0) / 1000.0 AS solar_kw FROM "East Microinverter" WHERE time >= now() - INTERVAL '1 minute' ORDER BY time DESC LIMIT 1
    ```
  - Query B (refId: "B"):
    ```sql
    SELECT COALESCE(power_sensor, 0) / 1000.0 AS solar_kw FROM "West Microinverter" WHERE time >= now() - INTERVAL '1 minute' ORDER BY time DESC LIMIT 1
    ```
  - Both with `format`: "table"
- `fieldConfig.defaults`:
  - `unit`: "kwatt"
  - `decimals`: 2
  - `color`: { "mode": "thresholds" }
  - `thresholds`: { "mode": "absolute", "steps": [{ "value": null, "color": "dark-gray" }, { "value": 0.001, "color": "yellow" }, { "value": 1, "color": "green" }, { "value": 4, "color": "dark-green" }] }
- `options`:
  - `reduceOptions`: { "calcs": ["lastNotNull"] }
  - `graphMode`: "none"
  - `textMode`: "auto"
  - `colorMode`: "value"
- Add a Grafana **transformation** to sum the two query results:
  ```json
  "transformations": [
    { "id": "merge", "options": {} },
    { "id": "reduce", "options": { "reducers": ["sum"] } }
  ]
  ```
  
  **Alternative approach (simpler — preferred):** Use a single UNION ALL query instead of two separate queries + transformations:
  ```sql
  SELECT SUM(solar_kw) AS solar_kw FROM (SELECT COALESCE(power_sensor, 0) / 1000.0 AS solar_kw FROM "East Microinverter" WHERE time >= now() - INTERVAL '1 minute' ORDER BY time DESC LIMIT 1 UNION ALL SELECT COALESCE(power_sensor, 0) / 1000.0 AS solar_kw FROM "West Microinverter" WHERE time >= now() - INTERVAL '1 minute' ORDER BY time DESC LIMIT 1)
  ```
  Use this single-query approach with refId "A", format "table". No transformations needed.
